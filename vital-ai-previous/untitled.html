Demonstrations of using the API:

Hello World Example

A simple example application is provided to get you started using the Vital AI Platform.  In this example: the base data model is extended with a new class, a simple Natural Language Processing rule is added to extract new entities, and the example code calls the service to process some text and print out the results.  Details are found here:

http://publicwiki.vital.ai/hello-world-example

Grails Demonstration Application

An example application using Vital AI Platform with the Grails Web Framework is available with full source code as a "starter" application.  Details are found here:

http://publicwiki.vital.ai/grails-news-portal

Grails Dashboard Demonstration Application

Following on the previous example application, a dashboard is used to monitor the activity of an application, and to tune it on an ongoing basis.  This example implements a simple dashboard for the previous application, also using the Grails Web Framework.  The full source code is available as a "starter" dashboard.  Details are found here:

http://publicwiki.vital.ai/grails-news-dashboard

Adding Recommendations to an Existing Application

The Vital AI Platform can be used to provide recommendations in an existing application.  This example shows the steps needed to integrate the Vital AI platform to add new data-driven functionality.  Details are found here:

http://publicwiki.vital.ai/add-recommendations-app
