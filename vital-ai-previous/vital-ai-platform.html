Vital AI Platform

The Vital AI platform aligns your application data model with your back-end data for analysis using the VitalSigns data tools, greatly easing integration, data maintenance, and eliminating additional data mapping. 

The Vital AI platform includes Machine Learning, Natural Language Processing, Logical Inference, Graph Analytics, Data Analysis via Hadoop, Web Crawling, integration with Twitter and Facebook, and large scale semantic data management.  The plaform integrates easily with additional application logic and front ends via REST and Queue interfaces.
